This project is for a crypto investment backtesting app. It's designed to take
3 pieces of information and show how investments could have panned out with the given
information. The given information is intended to relay:

1. A graph representing the three values in a 6 month periood.
2. The buy and sell points at when the user could have entered and exited their position.
3. The returns the user could have had in percentage form

Our 3 goals for this app was to:
1. Showcase our ability to work with one or more 3rd party APIs
2. Authenticate information sent and recieved from api 
3. Automate data manipulation and analasys
